# Python Interview Prep

This repository contains my structured preparation for Python coding interviews. It includes problem-solving approaches, key concepts, and curated practice questions aimed at strengthening core programming and algorithmic skills.

---

## 📌 Objectives

* Strengthen Python fundamentals
* Practice data structures and algorithms
* Prepare for coding interviews (DSA + problem-solving)
* Build consistency in solving problems

---

## 🧠 Topics Covered

* Python Basics

  * Data types, loops, functions
  * List, tuple, set, dictionary operations

* Data Structures

  * Arrays / Lists
  * Strings
  * Stacks & Queues
  * Linked Lists
  * Trees & Graphs
  * Hashing

* Algorithms

  * Searching (Binary Search, Linear Search)
  * Sorting (Merge Sort, Quick Sort, etc.)
  * Recursion & Backtracking
  * Dynamic Programming
  * Greedy Algorithms
